Open-source AI agent platform. Build and manage multiple AI agents from a visual console. Embed them in any web app with widgets, navigation, and full UI control — self-hosted, MIT licensed.

Everything you need to ship AI agents

A complete platform — not just an API wrapper. Multi-agent management, visual designer, UI widgets, knowledge base, and embeddable SDK in one package.

🧠

AI Agent Engine

LangGraph-powered agents with 16 node types. Attack guard, RAG knowledge retrieval, MCP tool calling, fact memory, and streaming responses. 162 tests, battle-tested.

🎨

Visual Designer

Drag-and-drop agent graph builder. 16 node types on a React Flow canvas. Real-time YAML preview, AI-assisted natural language commands, and one-click deploy.

🔬

Analysis Pipeline

11 parallel LLM phases analyze every message before response — extracting memories, impressions, and intent, then dynamically adjusting response tone and precision.

📚

Knowledge Base (RAG)

Upload documents, FAQs, and policies. Vibeful chunks, embeds, and indexes them with pgvector. Agents automatically search your knowledge for accurate, grounded answers.

💬

Multi-Agent Console

Manage multiple agents from one dashboard. Each agent gets its own graph, MCP tools, knowledge contexts, and configuration. Switch between agents with a dropdown.

📊

Embeddable SDK + Widgets

Drop agents into any web app in 3 lines of code. Agents can render widgets — buttons, cards, forms, charts — and drive host app UX via the command protocol.

📊

Built-in Operations

Version history with diff viewer. A/B testing with statistics. Per-node performance monitoring. Token credit budgeting. All managed from a single console.

Who it's for

Three integration tiers, one platform.

💬

Passive Embed

Drop a chat widget into your SaaS. Agents answer questions from your docs via RAG. 3 lines of code. Perfect for support bots and knowledge assistants.

🎯

Active Embed

Agents drive your app UX — navigating views, opening modals, updating state. Your app registers command handlers; agents orchestrate workflows.

🏗️

Agent-Native Apps

Build entire applications where the agent IS the shell. Compose widgets, routes, and behaviors visually. No host app needed — the agent is the platform.

Get started

One command. No sysadmin knowledge needed.

git clone https://github.com/SimonJamesOdell/vibeful.git cd vibeful bash scripts/setup.sh

The setup script detects your system, installs missing packages,
and starts everything. No Docker required.
