Platform features

Everything to ship AI agents

A complete platform — not just an API wrapper. Multi-agent management, visual designer, UI widgets, knowledge base, and embeddable SDK in one package.

🧠

AI Agent Engine

LangGraph-powered agents with 16 node types. Attack guard, RAG knowledge retrieval, MCP tool calling, fact memory, and streaming responses.

LangGraph · DeepSeek
🎨

Visual Designer

Drag-and-drop agent graph builder. 16 node types on a React Flow canvas. Real-time YAML preview, AI-assisted commands, and one-click deploy.

React Flow · YAML
🔬

Analysis Pipeline

11 parallel LLM phases analyze every message before response — extracting memories, impressions, and intent, then dynamically adjusting tone.

11 parallel phases
📚

Knowledge Base (RAG)

Upload documents, FAQs, and policies. Vibeful chunks, embeds, and indexes them with pgvector. Agents automatically search for grounded answers.

pgvector · Embeddings
💬

Multi-Agent Console

Manage multiple agents from one dashboard. Each agent gets its own graph, MCP tools, knowledge contexts, and configuration.

Multi-tenant
📦

Embeddable SDK

Drop agents into any web app in 3 lines of code. Agents render widgets — buttons, cards, forms, charts — and drive host app UX via commands.

React · Script Tag · REST
📊

Built-in Operations

Version history with diff viewer. A/B testing with statistics. Per-node performance monitoring. Token credit budgeting. All in one console.

A/B Testing · Analytics
🔗

MCP Tool Calling

Agents call external tools via the Model Context Protocol. Connect agents to APIs, databases, or services — natively, without glue code.

MCP Protocol
🛡️

Attack Guard

Built-in prompt injection detection and guardrail nodes block adversarial inputs before they reach the LLM — production-ready safety out of the box.

Security · Guardrails
🎯

Headless API

Invoke agents programmatically from any backend. REST endpoints for execute and SSE streaming. Full response includes tool calls, token usage, and session data.

REST · SSE · JSON
🐍

Python SDK

pip install vibeful — first-class async client. client.execute() for headless invocation, client.stream() for real-time streaming. Typed with dataclasses.

Async · Typed · httpx
📄

Agent Pages

Agents create and publish pages with interactive widgets — forms, charts, tables, and cards. Markdown editor with auto-save, slug routing, and publish/unpublish.

Widgets · Markdown · Routing
🔄

Widget Event Loop

Users interact with widgets → events reach the agent → agent processes and responds → page re-renders with new widgets. Fully agent-driven applications.

Interact · Process · Render
🔑

API Keys & Auth

SHA-256 hashed keys with vf_ prefix. Scoped permissions per agent. Bearer token auth. Keys shown once — revoke anytime.

Hashed · Scoped · Revocable
👥

Users & Teams

Registration, login with password hashing, and team management built in. Add members, assign roles, scope agents to teams.

Auth · Roles · Multi-tenant
🔔

Webhooks

Subscribe to conversation.completed events. Fire-and-forget delivery with 10-second timeout. Build reactive workflows — Slack notifications, CRM updates, audit trails.

Events · Async · Best-effort

Integration tiers

Who it's for

Three integration tiers, one platform. From a script tag to agent-driven applications.

01
💬

Embed

Drop a chat widget into any page with 3 lines of code. Agents answer from your docs via RAG. Voice I/O, CSS theming, widget rendering.

  • <script> tag — no build step
  • RAG over your docs
  • Voice input & output
  • CSS custom properties
02
🔗

Integrate

Headless agent invocation from any backend. Streaming, webhooks, and SDKs for Python and JavaScript. Agents drive business logic, not just chat.

  • REST API + SSE streaming
  • Python SDK (pip install vibeful)
  • JS hooks (useAgent, useAgentStream)
  • Webhooks (conversation.completed)
03
🏗️

Agent-Native

Agents create and publish interactive pages with forms, charts, tables, and cards. Users interact → agent processes → pages update. No host app needed.

  • 8 widget types (form, chart, table, card…)
  • Widget event loop (interact → agent → re-render)
  • Page CRUD + slug routing
  • Import/export (.vibeful.yaml)
Get started

One command. No sysadmin needed.

The setup script detects your system, installs missing packages, and starts everything — no Docker required.

bash
# Clone the repo
git clone https://github.com/SimonJamesOdell/vibeful.git
cd vibeful
# Run the setup script — detects your system automatically
bash scripts/setup.sh

Full control from any backend. Create sessions, send messages, receive streaming responses.
