What Vibeful Gives You

🧠 AI Agents

Configure personality, knowledge, and tools through a no-code admin panel. Agents use RAG to answer from your documents, MCP to call external tools, and memory to remember users across sessions.

📚 Knowledge Base

Upload documents, FAQs, and policies. Vibeful chunks, embeds, and indexes them. Agents automatically search your knowledge to give accurate, grounded answers.

🔌 Tool Integration

Connect agents to search engines, databases, APIs, and any external system using the MCP protocol. Build custom tool servers in minutes with our framework.

📊 Analytics & Trust

Know what users ask, where knowledge gaps are, and how much you're spending. Built-in dashboards for usage, cost, and conversation quality.

💬 Embeddable SDK

Drop a 3-line code snippet into any web app. React component, HTML script tag, or REST API — pick the integration that fits your stack.

🔒 Self-Hosted

Your data stays on your infrastructure. No vendor lock-in. MIT licensed. Runs anywhere Docker runs — your laptop, a VPS, or a Kubernetes cluster.

5-Minute Quickstart

  1. Clone and configure
    git clone https://github.com/vibeful/vibeful.git
    cd vibeful
    cp .env.example .env
    # Add your DeepSeek API key to .env
  2. Start Vibeful
    docker compose up -d
  3. Open the admin panel
    # Go to http://localhost:5173
  4. Create your first agent
    # Agents tab → Name: "Support Agent" → Create
  5. Embed in your app
    <div id="vibeful-chat" style="max-width:400px;height:500px"></div>
    <script src="vibeful-sdk.umd.js"></script>
    <script>
    VibefulSDK.mount({ target: '#vibeful-chat', agentId: 'YOUR_AGENT_ID' });
    </script>

"Use Vibeful to add agentic functionality to my site"

That's it. Tell your AI coding agent that sentence, and it can pull Vibeful, configure an agent, and embed it — all from the docs and API reference in this repo.
