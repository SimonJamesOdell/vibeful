Get Started

5-Minute Quickstart — No File Editing

Clone, start, paste your API key in the browser. No .env files, no manual configuration.

  1. Clone and start
    # Clone the repo git clone https://github.com/vibeful/vibeful.git # Start everything (PostgreSQL, Redis, Agent Engine, Proxy, Console, SDK) cd vibeful && docker compose up -d
  2. Open the console and paste your API key
    # Open http://localhost:5174 # The setup wizard appears automatically # Paste your DeepSeek API key → click Save # Free keys at https://platform.deepseek.com/api_keys
  3. The Vibeful Guide greets you
    # "Welcome! I see this is your first time. # Would you like me to help you build your first agent?" # Just say yes — the Guide sets up your agent on the canvas.
  4. Deploy and embed
    # Click Deploy — your agent goes live, get an Agent ID # Embed in your app with 3 lines: <div id="vibeful-chat"></div> <script src="vibeful-sdk.umd.js"></script> <script>VibefulSDK.mount({target:'#vibeful-chat',agentId:'YOUR_ID'})</script>
Integration

Three ways to embed

Pick the integration that fits your stack. All three use the same agent engine and API.

🏷️

HTML Script Tag

Drop into any website. No build step. Customizable with CSS variables for theming.

<div id="chat"></div> <script src="sdk.js"></script> <script>VibefulSDK.mount({ target:'#chat', agentId:'YOUR_ID' })</script>
⚛️

React Component

First-class React support with hooks. Full TypeScript types. Command protocol for UI control.

import { VibefulChat } from '@vibeful/sdk'; function App() { return <VibefulChat agentId="YOUR_ID" theme={{ '--user-bg': '#7c3aed' }} />; }
🔗

REST API

Full control from any backend. Create sessions, send messages, receive streaming responses with citations and follow-ups.

// Create session POST /v1/sessions { "agent_id": "YOUR_ID" } // Send message POST /v1/sessions/:id/converse { "content": "Hello!" }
Agent Inception

An agent that helps you build agents

The Vibeful Guide lives inside the management console. It demonstrates the product by being the product — using the same analysis pipeline, command protocol, and RAG that your agents will use.

🧭 Onboarding Assistant

Detects empty canvas → auto-opens with a welcome message. Walks new users through building their first agent step by step. Answers questions about Vibeful concepts, architecture, and best practices.

🖱️ Canvas Control

Issues structured commands that add nodes, connect edges, deploy agents, and switch tabs. The same Vibeful Command Protocol is available to your agents — letting them navigate, open modals, and drive your UI.
